# SpectreWall's Stack Refill

A lightweight plugin that automatically refills your hand stack when placing blocks.
(if more of the same block is available in your hotbar, inventory or backpack)

> ✨ **Ready to build immediately!**

Clone the repository and run:

```bash
./gradlew build
```

The compiled plugin JAR will be generated in the `build/libs` directory.
